# NutriHub
ECE 4574: Group D

## Setting up the Dev Environment
Create a virtual environment
```
python3 -m venv .venv
```
Activate Environment
```
source .venv/bin/activate
```
Install requirements
```
pip install -r requirements.txt
```
Run the app
```
flask --app main run
```
